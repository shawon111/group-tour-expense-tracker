# TripTrack

TripTrack is a lightweight travel expense tracker for teams. Track group expenses, see who paid what, and keep trip finances transparent.

## Features
- Add, edit, and delete expenses
- Per-user and team totals
- Team overview with member breakdown
- Simple authentication (Supabase)
- Mobile-first responsive UI

## Quick start

Prerequisites:
- Node.js (or Bun)
- A Supabase project (for database & auth)
- Environment vars in `.env`

Install dependencies and run dev:
npm install
npm run dev
